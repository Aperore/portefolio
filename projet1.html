Projet applicatif CNED

Ajouter des fonctionnalités à une médiatèque de gestion de formations

Description du projet

Au cours de mon BTS avec le CNED, dans le cadre d'une activité, ce projet a été mis en place pour mettre en pratique les méthodes apprises au cours de ces deux années.

Il s'agissait de nettoyer le code, d'ajouter des fonctionnalités (ajouter, modifier, trier) et de mettre en place le back office. Des tests unitaires, d'intégration et fonctionnels ont aussi été effectués. Lancer le déploiement du site. Une documentation technique est disponible.

Cela m'a permis de voir concretement comment toute ces étapes se déroulent tout en étant acteur dans l'évolution du projet.

Technologies utilisées

GitHub NetBeans PHP MySQL
